# QA-Manual-Testing-Portfolio
Project Overview
This project involves comprehensive manual testing of the OrangeHRM open-source application. I developed over 80+ test scenarios and detailed test cases to ensure system reliability and a seamless user experience.



Key Activities
* Test Documentation: Created detailed Test Cases for Login, Recruitment, Performance, and Attendance modules. * Execution: Performed functional and negative testing for robust data validation. * Defect Tracking: Documented functional defects with High/Medium severity using a structured report format. * Workflow Validation: Verified critical paths like "Forgot Password" and search filters.




Tools Used
* Methodology: Agile. * Tools: Jira for defect tracking and Microsoft Excel for documentation.
